# Demo Website

A sample multi-directory website built using the **git-workflow scripts**.

This repository was created automatically by `demo-workflow.sh` to demonstrate
the full Git workflow including branching, conventional commits, pull requests,
syncing, and cleanup.

## Structure

```
├── index.html          - Main landing page
├── css/styles.css      - Stylesheet
├── js/main.js          - JavaScript
└── pages/
    ├── about.html      - About page
    └── contact.html    - Contact page
```
